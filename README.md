### ⚡ The Vibe Coding Workflow

My development environment is built on **Fedora Linux**, optimized for extreme speed and heavy automation:
* **Assembling over Writing:** I orchestrate multi-agent setups using **Cursor** and the **OpenCode Agent** to handle codebase generations in seconds.
* **Terminal-First Mentality:** Complex environments, environment variables, and deployment pipelines are managed directly via a customized Bash terminal.
* **Strict Validation:** I treat ИИ outputs with professional skepticism — debugging architectural limitations, managing context windows, and refining prompts until the system runs flawlessly.
